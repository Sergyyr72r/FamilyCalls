# Agora.io Video Calling Setup

## What is Agora.io?

Agora.io provides a video calling SDK that works within your app - no external apps needed!

**Benefits:**
- ✅ Free tier: 10,000 minutes/month free
- ✅ Easy to integrate
- ✅ Available in Maven (no manual downloads)
- ✅ Works within your app
- ✅ Good quality video/audio

## Setup Steps

### Step 1: Create Agora Account (Free)

1. Go to: https://www.agora.io/
2. Sign up for free account
3. Create a new project
4. Get your **App ID** and **App Certificate** (optional)

### Step 2: Add App ID to Your App

Once you have your App ID, I'll add it to the code.

### Step 3: The Code is Already Updated!

I've already:
- ✅ Added Agora SDK to `build.gradle`
- ✅ Updated `VideoCallActivity` to use Agora
- ✅ Set up video calling functionality

### Step 4: Get Your Agora App ID

1. Login to Agora Console: https://console.agora.io/
2. Create a project (or use existing)
3. Copy your **App ID**
4. Share it with me and I'll add it to the code
